# Система учёта заявок на ремонт бытовой техники

Веб-приложение для управления заявками на ремонт бытовой техники, разработанное на FastAPI.

## Описание

Система позволяет:
- Клиентам создавать заявки на ремонт бытовой техники
- Операторам обрабатывать и редактировать заявки
- Мастерам добавлять комментарии к заявкам
- Менеджерам назначать мастеров на заявки
- Администраторам управлять пользователями и просматривать статистику

## Технологии

- FastAPI
- SQLAlchemy
- SQLite
- Jinja2 Templates
- HTML/CSS

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python -m app.main
```

Приложение будет доступно по адресу: http://127.0.0.1:8000
