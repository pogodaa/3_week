# Протокол тестирования Задания 1

## Тестируемая функция
`calculate_avg_repair_time()` — расчёт среднего времени выполнения заявок.

## Тестовый сценарий
Загрузить данные из файла `inputDataRequests.xlsx` (неделя 2).
Отфильтровать заявки со статусом «Готова к выдаче».
Завершённые заявки:
- №3: `startDate` = 2022-07-07, `completionDate` = 2023-01-01 → 178 дней
- №6: `startDate` = 2023-08-02, `completionDate` = 2023-08-03 → 1 день
- №7: `startDate` = 2023-07-09, `completionDate` = 2023-08-03 → 25 дней  
Среднее = (178 + 1 + 25) / 3 = **68.0**

## Ожидаемый результат
`68.0`

`68.0` (тест `pytest` → PASSED)
